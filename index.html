Why Manager

Watch the fleet — and drive the engine that improves it.

Six things that turn a family of self-improving repos into a calm, legible command center you actually control.

A living dashboard

One tile per project: a health score, the latest version, the time it last shipped, a velocity sparkline, and a short assessment. It updates the moment anything changes.

Fleet Ops: schedule & run it

Turn each app's self-improvement loop on or off, set its cadence and time window, dispatch one-off runs, and preview exactly what fires next — the whole platform steward, driven from here.

The Steward log

Every self-improvement run, reviewable in place — the steward's own account of what it shipped, the safety nets that keep every site green, and open work across the whole fleet.

A curated library

Filter, sort, search, pin, and edit every project. Add your own typed metadata fields, save the views you use most, and rename or clean up tags fleet-wide.

Every "what's new"

Each project keeps its own release timeline, synced live from its deployed changelog. See the latest and the whole fleet's history, searchable and sortable.

Local-first, yours to keep

Works fully in your browser, or connect a real database (Turso, Supabase, Firebase) to reach it anywhere — with an option to encrypt your secrets at rest. Export, import, undo.

How it works

From scattered repos to one self-shipping fleet.

Add your projects

Point Manager at each repo and site, drop in the Claude Code session URL, and tag it however you think. Manager already knows the whole polecat.live family — including itself.

Schedule the steward

From Fleet Ops, give each app a cadence — hourly, every few hours, or fenced to a window. A shared GitHub Actions steward builds one improvement at a time, tests it, and merges only when green.

Watch it ship — and review

The dashboard shows what shipped and when; the Steward log shows what each run actually did. Jump into any session, connect a token to run things yourself, and keep the library sharp.

Invite-only

Locked down, by design.

Manager is a private preview. Access is a signed, unforgeable token verified entirely in your browser — no server, no accounts. Admins hold a token that also mints shareable invite links.

Admin token

Full access plus the power to mint invites. Paste it once on the unlock screen.

Invite links

Share a signed /app/?invite=… link — it unlocks the app for whoever opens it.

Revocable

Set an expiry or revoke a link any time from the in-app Admin area.
