Why Manager

Everything your fleet is doing — at a glance.

Six things that turn a pile of self-improving repos into a calm, legible command center.

A living dashboard

One tile per project: status, the time it last shipped in CT, the latest version, and a short assessment. It updates the moment anything changes.

A curated library

Filter, sort, search, pin, and edit every project. Add your own metadata fields and save the views you use most.

Every "what's new"

Each project keeps its own release timeline. See the latest and the whole history, searchable and sortable.

Straight to the session

One click opens the Claude Code session steering each project — no more hunting for the right tab.

Cadence you can see

Manager tracks the feature/sweep rhythm — four features, then a polish sweep — so you always know what's next.

Yours, and portable

Local-first storage, one-click export/import, history + undo. Shaped like a database so it can graduate to SQLite later.

How it works

From scattered repos to one console.

Add your projects

Point Manager at each repo and site, drop in the Claude Code session URL, and tag it however you think.

Let them self-improve

Each project runs its own loop — a cron routine or a GitHub Action — shipping features and a changelog on a schedule.

Watch, steer, curate

The dashboard shows what shipped and when. Jump into any session, log the cadence, and keep the library sharp.

Invite-only

Locked down, by design.

Manager is a private preview. Access is a signed, unforgeable token verified entirely in your browser — no server, no accounts. Admins hold a token that also mints shareable invite links.

Admin token

Full access plus the power to mint invites. Paste it once on the unlock screen.

Invite links

Share a signed /app/?invite=… link — it unlocks the app for whoever opens it.

Revocable

Set an expiry or revoke a link any time from the in-app Admin area.

Enter with your token

See your whole fleet in one place.

It runs the moment it loads — paste your token and go.
